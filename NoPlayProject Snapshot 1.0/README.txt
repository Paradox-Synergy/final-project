La versione corrente contiene tutte le funzionalità di lettura, mancano ancora le funzioni di modifica, aggiunta ed eliminazione.

Per utilizzare il programma, eseguire tramite un IDE il file AigiApplication.java e aprire la pagina Home, dopo aver impostato correttamente le credenziali di accesso al database nel file application.properties

È consigliato disporre di un DBMS per creare il database "noplay" e inserire i dati iniziali, le tabelle saranno generate all'avvio del programma Java.
